To start using the bot, you need to pair it with your WhatsApp account and add it to a group.

Pair Device

Check the terminal logs for the pairing code or QR code. Go to WhatsApp > Linked Devices > Link a Device.

2

Add to Group

Add the bot's phone number to any WhatsApp group you want to protect.

3

Grant Consent

The bot will not scan messages until an admin grants consent. Run this command in the group:

!scanner consent
