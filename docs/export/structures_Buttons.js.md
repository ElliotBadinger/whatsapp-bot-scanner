     * Creates button array from simple array

     * @returns {FormattedButtonSpec[]}
     * @example 
     * Input: [{id:&#x27;customId&#x27;,body:&#x27;button1&#x27;},{body:&#x27;button2&#x27;},{body:&#x27;button3&#x27;},{body:&#x27;button4&#x27;}]
